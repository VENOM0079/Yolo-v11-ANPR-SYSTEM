# Models Directory

Place your YOLOv11 model weights here.

## Download Models

```bash
# Download YOLOv11n (nano - fastest)
python -c "from ultralytics import YOLO; YOLO('yolov11n.pt')"

# Or use larger models for better accuracy:
# yolov11s.pt - small
# yolov11m.pt - medium
# yolov11l.pt - large
# yolov11x.pt - extra large
```

## Model Files

- `yolov11n.pt` - Default model for vehicle detection
- Custom trained models can also be placed here
